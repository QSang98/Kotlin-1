* class:
  class trong kotlin được khai báo với từ khóa class
  VD:
  class SinhVien {
  } 

* abstract class:
  abstract class trong kotlin được khai báo với từ khóa abstract trước class

  VD:
  
  abstract class Hinh {
  }

* interface:
  interface trong kotlin được khai báo với từ khóa interface
  
  VD:
  
  interface Dog {
  }
